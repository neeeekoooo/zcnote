## 表示法
- - console打印出来都是10进制,除非指定输出格式
- go程序不能直接二进制表示一个数字,可以用`fmt.Printf("%b",num)`可以查看二进制
- 八进制(0到7,满8进1,数字0开头)
- 十六进制(0到9,A到F,满16进1,数字0x开头)

## 进制转换
```
# 二进制转十进制
1011 -> 1*2^0 + 1*2^1 + 0*2^2 + 1*2^3 = 11

# 八进制转十进制
0123 -> 3*8^0 + 2*8^1 + 1*8^2

# 十六进制转十进制
0x34A -> 10*16^0 + 4*16^1 + 3*16^2 = 842
```
