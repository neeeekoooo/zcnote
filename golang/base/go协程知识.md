### 1. 协程执行顺序

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	go func() {
		for i := 0; i < 10; i++ {
			fmt.Println("a")
		}
	}()

	go func() {
		for i := 0; i < 10; i++ {
			fmt.Println("b")
		}
	}()

	go func() {
		for i := 0; i < 10; i++ {
			fmt.Println("c")
		}
	}()

	time.Sleep(30 * time.Second)
}
```


这说明了一旦创建一个协程，代码不一定会协程中的代码，各个协程执行顺序由系统调用，没有先后的说法，同一时间只能处理一个协程代码，不是多线程。 协程不会并行执行，是协同执行
